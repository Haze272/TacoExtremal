# SPA по доставке мексиканской еды на Angular
## Вступление
Данный репозиторий содержит в себе одностраничное веб-приложение, написанное на Angular, являющиеся реализацией сервиса по доставке мексиканской еды. Приложение хранит страницы товаров, корзину и оформление заказа.
## Установка
Для установки склонируйте репозиторий, и, находясь в папке проекта напишите следующие команды:
`npm install` - для установки зависимостей
`npm run start` - для локального запуска приложения на хосте 4200
